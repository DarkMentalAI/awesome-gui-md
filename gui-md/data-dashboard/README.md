# Data Dashboard

Dense operational dashboard pattern for metrics, filters, tables, and bulk actions.

## Why It Matters

Dashboards fail when they only show the happy path. A useful dashboard needs clear loading, empty, error, filtered, selected, and permission-limited states.

## Files

- [GUI.md](GUI.md) - Interface contract.
- [HTML.md](HTML.md) - Web adapter contract.
- [meta.json](meta.json) - Index metadata and coverage summary.

## Good Fit

- analytics products
- admin tools
- monitoring surfaces
- finance and operations workflows

## Bad Fit

- simple marketing analytics summaries
- mobile-first flows with very small data sets
